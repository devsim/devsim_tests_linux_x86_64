Windows 64 bit with Anaconda Python 2.7 and Intel MKL

Requirements:
Python 2.7 (Miniconda installed in c:\Miniconda-x64)
Miniconda packages (conda install numpy mkl)
Cygwin 32 bit
CMake from http://www.cmake.org
